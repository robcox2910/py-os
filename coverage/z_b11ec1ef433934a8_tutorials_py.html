1"""Interactive tutorial system for learning OS concepts.

3Each lesson is a guided walkthrough that uses **real syscalls** to 

4demonstrate a subsystem. The goal is to make the OS feel tangible: 

5you see processes being created, memory being allocated, files being

10 

111. Opens with a **real-world analogy** (recipes, warehouses, libraries…).

76 Raises:

97 def run_all(self) -> str:
