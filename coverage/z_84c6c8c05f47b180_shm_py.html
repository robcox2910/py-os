1"""Shared memory IPC — named memory regions for inter-process communication. 

2 

3Shared memory is the fastest IPC mechanism. Unlike pipes (streaming, 

4one-way) or message queues (discrete messages), shared memory lets 

5multiple processes read and write the same underlying bytearrays with 

6zero copying. Unlike mmap (file-backed), shared memory is 

7**anonymous** — no associated file, identified only by name. 

8 

9Think of it like a shared whiteboard in the school hallway. Any 

10student can walk up and write on it, and everyone else can see what's 

11there instantly. But if two students try to write at the same time, 

12their writing might overlap — so they need a system (like taking turns 

13with a semaphore). 

14""" 

15
